Tutorial Pengisian

Panduan lengkap cara mengisi dan mengupload data sekolah

1

Unduh Template Excel

Kunjungi menu "Upload Excel" lalu klik tombol "Unduh Template" untuk mendapatkan file Excel kosong yang sudah ada formatnya.

2

Isi Data di Template

Buka file Excel yang sudah diunduh, lalu isi data sesuai kolom yang tersedia. Pastikan tidak mengubah header kolom.

3

Upload File Excel

Kembali ke menu "Upload Excel", drag & drop atau pilih file Excel yang sudah diisi. Preview data akan muncul sebelum diupload.

4

Verifikasi Data

Setelah upload berhasil, kunjungi menu "Data Tersimpan" untuk memastikan seluruh data sudah masuk dengan benar.
