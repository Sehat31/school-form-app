Tutorial Pengisian

Panduan lengkap cara menggunakan Portal SPP Gjatian

1

Isi Data Sekolah

Buka menu "Data Sekolah", isi form dengan lengkap, lalu klik "Simpan & Lanjutkan ke MBG".

2

Unduh Template MBG

Di menu "Data MBG", klik "Unduh Template MBG" untuk mendapatkan file Excel dengan 2 sheet (PM dan Guru & Tendik).

3

Isi Template Excel

Buka file Excel, isi data di sheet PM untuk siswa dan sheet Guru & Tendik untuk guru. Jangan ubah header kolom.

4

Upload File Excel

Kembali ke menu "Data MBG", pilih sekolah, lalu drag & drop file Excel. Preview data akan muncul sebelum disimpan.

5

Verifikasi Data

Buka menu "Data Tersimpan" untuk melihat semua data yang sudah masuk. Gunakan fitur search dan filter untuk mencari data.

Pertanyaan Umum (FAQ)

File yang didukung adalah format .xlsx dan .xls dengan ukuran maksimal 10MB.

Untuk data sekolah bisa langsung via form. Untuk data MBG (Siswa & Guru) saat ini hanya via upload Excel.

Anda bisa mengedit atau menghapus data melalui menu "Data Tersimpan", lalu klik ikon edit/hapus pada baris yang bersangkutan.
