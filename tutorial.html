Pusat Tutorial

Panduan lengkap menggunakan untuk manajemen data sekolah dan program MBG

🎯

Apa itu FORM DATA ?

FORM DATA adalah sistem manajemen data sekolah terintegrasi untuk mengelola:

  • Data Sekolah - Informasi lengkap sekolah
  • Data MBG - Penerima Manfaat (Siswa) program Makan Bergizi Gratis
  • Data Guru & Tendik - Tenaga pendidik dan kependidikan

Alur Kerja Lengkap

Ikuti 5 langkah sederhana ini untuk mulai menggunakan sistem:

1. Isi Data Sekolah

Lengkapi formulir data sekolah dengan informasi yang akurat

2. Download Template

Unduh template Excel untuk data MBG dan Guru

3. Isi Data Excel

Masukkan data siswa dan guru ke dalam template

4. Upload File

Unggah file Excel yang sudah diisi ke sistem

5. Lihat Data

Periksa dan kelola data yang sudah tersimpan

Tips Cepat

Gunakan menu navigasi di atas untuk melompat ke bagian tutorial yang ingin Anda pelajari. Setiap bagian memiliki panduan langkah demi langkah dengan gambar ilustrasi.

1

Mengisi Data Sekolah

Langkah pertama adalah mengisi formulir data sekolah. Pastikan semua informasi yang dimasukkan akurat dan lengkap.

Field yang Harus Diisi:

Jenjang

Pilih tingkat pendidikan: TK, SD, SMP, SMA, atau SMK

Nama Sekolah

Masukkan nama lengkap sekolah sesuai akta

Nama Kepala Sekolah

Nama lengkap kepala sekolah beserta gelar

NPSN

Nomor Pokok Sekolah Nasional (8 digit angka)

Alamat Sekolah

Alamat lengkap dengan kode pos

Nama PIC

Person In Charge yang bisa dihubungi

Nomor HP PIC

Nomor telepon yang aktif (10-15 digit)

SPP Bulanan

Biaya SPP per bulan (angka saja, tanpa Rp)

Contoh Pengisian

Jenjang: SD
Nama Sekolah: SDN 1 Menteng
NPSN: 20100123
SPP Bulanan: 500000

Setelah selesai mengisi, klik tombol "Simpan & Lanjutkan ke MBG" untuk melanjutkan ke tahap berikutnya.

2

Upload Data MBG (Makan Bergizi Gratis)

Setelah data sekolah tersimpan, lanjutkan dengan upload data Penerima Manfaat (siswa) dan Guru & Tendik.

Langkah 1: Download Template

Klik tombol "Unduh Template MBG" untuk mendapatkan file Excel dengan 2 sheet:

  • PM (Penerima Manfaat) - untuk data siswa
  • Guru & Tendik - untuk data guru dan tenaga kependidikan

Langkah 2: Isi Template Excel

Buka file Excel yang sudah diunduh dan isi data sesuai kolom yang tersedia.

Sheet PM (Penerima Manfaat):

NIK (16 Digit)

Nomor Induk Kependudukan siswa

NISN (10 Digit)

Nomor Induk Siswa Nasional

Nama Lengkap

Sesuai Akta/KTP

Tempat Lahir

Kota/Kabupaten

Tanggal Lahir

Format: dd-mm-yyyy (contoh: 01-01-2015)

Jenis Kelamin

L (Laki-laki) atau P (Perempuan)

Nama Orang Tua/Wali

Nama ayah/ibu/wali siswa

Kelas

Contoh: 1, 7, 10

Sheet Guru & Tendik:

NIK (16 Digit)

Nomor Induk Kependudukan guru/staf

Nama Lengkap

Sesuai KTP

Tempat & Tanggal Lahir

Tempat lahir dan tanggal lahir (dd-mm-yyyy)

Jenis Kelamin

L atau P

Jabatan

Guru atau Tendik (Tenaga Kependidikan)

Penting!

JANGAN ubah nama kolom/header di template Excel. Sistem membutuhkan header yang tepat untuk memproses data.

Langkah 3: Upload File Excel

Setelah file Excel selesai diisi, kembali ke menu "Data MBG" dan upload file.

  1. Pilih Sekolah - Pilih sekolah dari dropdown NPSN
  2. Drag & Drop - Seret file Excel ke area upload, atau klik "Pilih File"
  3. Preview Data - Sistem akan menampilkan preview data yang akan diupload
  4. Klik Upload - Klik tombol "Upload & Simpan ke Database"
Validasi Otomatis

Sistem akan memvalidasi data sebelum disimpan. Pastikan tidak ada error sebelum upload!

Fitur-Fitur Lengkap

FORM DATA dilengkapi dengan berbagai fitur untuk memudahkan pengelolaan data.

Pencarian Data

Cari data sekolah, siswa, atau guru dengan cepat menggunakan kolom pencarian. Sistem akan memfilter hasil secara real-time.

Filter Berdasarkan Kategori

Filter data berdasarkan jenjang, kelas, atau kategori lainnya untuk melihat data yang lebih spesifik.

Export Data

Download data dalam format Excel untuk keperluan laporan atau backup. Klik tombol "Export" di halaman Data Tersimpan.

Hapus Data

Hapus data yang tidak diperlukan dengan klik ikon trash. Konfirmasi akan muncul sebelum data dihapus permanen.

Riwayat Upload

Lihat semua file Excel yang pernah diupload di bagian "File yang Sudah Diupload". Download atau hapus file kapan saja.

Statistik Data

Lihat ringkasan statistik: Total Sekolah, Total Penerima Manfaat, dan Total Guru & Tendik di dashboard.

Cara Melihat & Mengecek Data

  1. Klik menu "Data Tersimpan" di navigasi
  2. Pilih tab yang diinginkan:
    • Data Sekolah - Lihat semua data sekolah
    • Penerima Manfaat - Lihat data siswa
    • Guru & Tendik - Lihat data guru dan staf
  3. Gunakan kolom pencarian untuk mencari data spesifik
  4. Klik tombol "Export" untuk download data dalam format Excel
